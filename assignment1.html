Assignment 1

Write your name, a integer variable, a floating point variable, and double-precision variable on the screen. 


            program variables
                implicit none
                !Declare data types
                character(len=15) :: name
                integer :: int_var
                real :: float_var
                !double precision :: double_var
                real*8 :: double_var
                
                name = "Sumit Ghosh"
                int_var = 5
                float_var = 3.14
                double_var = 2.71828
                
                print *, "My name:",name
                print *, "A integer variable:",int_var
                print *, "A floating point variable:",float_var
                print *, "A double-precision variable:",double_var
                
            end program variables

Print 10 random numbers between 0 and 1 on your screen., using real*8 variable. 


            program random_numbers
                implicit none
                real*8 :: r
                integer :: i,seed
                seed = 100235
                call srand(seed)
                do i = 1, 10
                    r = rand()
                    write(*,*) r
                end do
            end program random_numbers
